## 💎 Hlavní funkce (Aktuální verze 2.0)

### 🚨 SOS Moduly (První pomoc)
Dvě specializované dechové animace pro okamžitou změnu fyziologie:
- **🔥 VZTEK:** Rychlejší 6sekundový cyklus (3s nádech / 3s výdech) pro vybití adrenalinu a zastavení křiku.
- **🌪️ ÚZKOST:** Pomalý 12sekundový cyklus s pauzou pro uzemnění při pocitu přehlcení.

### 🎯 Dynamické Mikro-kroky
Aplikace každé ráno vygeneruje unikátní dvojici úkolů:
- **Pro mě:** Preventivní seberozvoj (např. ticho v autě, studená sprcha).
- **Pro ni:** Praktická podpora manželky (např. čaj bez ptaní, aktivní naslouchání).
*Algoritmus rotuje 20 úkolů podle dne v roce.*

### 📝 Analytický Deník & Historizace
Pokročilé sledování trendů v domácnosti:
- **Sledování emocí:** Škála vzteku 0-5.
- **Identifikace spouštěčů:** Práce, hluk dětí, únava, komunikace.
- **Stav partnerky:** Monitoring jejího vyčerpání pro včasnou reakci.
- **Historizace:** Data se ukládají do `LocalStorage` prohlížeče (zůstávají v mobilu).

### 🛠️ Rychlé nástroje
- **🔈 Audio opora:** Smyčka uklidňujícího zvuku deště/bílého šumu.
- **❤️ Signál pro ni:** Tlačítko pro okamžité odeslání předpřipravené povzbuzující SMS.
- **🔋 Baterie odolnosti:** Vizuální tracker tvého aktuálního "streaku" bez výbuchů.
